## 如何使用

### 第一步：添加多个图集链接
在 **TelegramDownloader.java** 中的 **list** 中存放你要下载的telegram图集链接

### 第二步：设置你要下载存储的文件路径
在 **DownLoadUtil.java** 更改 **filePath** 为你的存放地址

### 最后一步：运行代码
程序会在你的 **filePath** 下创建需要下载的图集的文件夹并启动需要下载的图集数量的线程自动下载，下载速度取决于图集的大小和你的网速
